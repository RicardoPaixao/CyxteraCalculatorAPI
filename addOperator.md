#CyxteraCalculatorAPI

** AddOperator **
----
request used to add a new operator to this session's operator stack

* **URL**

CyxteraCalculator/webresources/web.operators
* **Method:**
  
POST  

* **Data Params**

`<operators>
    <sessionId>11</sessionId>
    <value>2</value>
</operators>`

* **Success Response:**
  
  * **Code:** 200 <br />
