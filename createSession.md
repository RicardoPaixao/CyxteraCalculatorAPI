#CyxteraCalculatorAPI

** CreateSession **
----
request used to create a new calculation session. is expected to be the first request of any interaction
  
* **URL**

CyxteraCalculator/webresources/web.sessions/createSession

* **Method:**
  
GET

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** `{ id : 12 }`
